# Work Progress Tracker

This folder is the live execution tracker for BrahmiBhojan.

## Purpose

Track what was built, how it was built, current progress, remaining TODOs, and edge cases to revisit.

## Update Protocol (after every completed task)

1. Add one entry in `task-log.md`.
2. Update module percentage and phase status in `development-status.md`.
3. Move related items in `todo-backlog.md` from Open to Done.
4. Add new edge cases or risks in `edge-cases-and-risks.md`.

## Files

- `task-log.md`: Chronological implementation log.
- `development-status.md`: High-level progress dashboard.
- `todo-backlog.md`: Prioritized TODO and done lists.
- `edge-cases-and-risks.md`: Edge scenarios, risks, and mitigations.

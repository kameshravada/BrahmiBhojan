# 27 - Coding Standards

Status: Draft for approval

## Backend

- Java 21.
- Spring Boot 3.
- Constructor injection.
- DTOs for API boundaries.
- Validation on inputs.
- Services own business logic.
- Transactions around consistency-sensitive operations.
- Flyway for schema changes.

## Frontend

- TypeScript strictness where practical.
- Feature-based components.
- Zod schemas for forms/payload validation.
- React Query for server state.
- Zustand only for client UI state.
- Accessible, responsive UI.

## Git

- Meaningful commits.
- No secrets.
- PR reviews before production-critical changes.

# 22 - Analytics

Status: Draft for approval

## Required Metrics

- Revenue
- Orders
- Average order value
- Customer lifetime value
- Repeat customers
- Cart abandonment
- Coupon usage
- Payment modes
- Top products
- Low-selling products
- Most viewed products
- Search keywords
- Conversion funnel
- Traffic sources
- Revenue by country/state/district/town/village/locality/pincode
- Delivery performance
- Cancellation reasons
- Return reasons
- Marketing campaign performance

## Event Strategy

Capture page view, product view, search, filter, add to cart, checkout start, payment start, payment result, order created, coupon applied, notification sent, review submitted.

# 02 - Routing and Layout

## Route Groups (planned)

- Public: home, catalog, product details
- Auth: OTP request/verify
- Protected: cart, checkout, profile, orders

## Layouts (planned)

- Public layout (header, category nav, footer)
- Auth layout (minimal shell)
- App layout (authenticated user navigation)

## Guarding Strategy

- Route guards for authenticated routes
- Token refresh handling at API layer
